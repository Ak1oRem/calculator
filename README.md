# Calculator
Several ways to build a calculator app from the most simple and light weight to more robust and feature hevy.
My priactice project to train proper project management and planning. 

# First iteration
Python only desktop app with minimal gui. 
Reqs:
Small window with buttons for digits, backspace, equal, history, and 4 arithmetic operations.
Results of a simple number on number calculation a shown in the text field along with the expression.
History button will use the whole window covering the rest of the buttons and show the results of all the previous times you pressed equal with it's results.
Pressing on an entry will enter it in to the input field.
You also can input using keyboard, and you can change the position of the cursor with arrow keys or mouse click between the values.
